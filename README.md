## Tracking Items
Right click an item in the bank or shared storage interface.<br><br>
<img width="551" height="69" alt="Item variant detection" src="https://github.com/user-attachments/assets/522526db-2dc7-43b0-ba0b-ce64c231ac22" />

## Variants
Items are tracked as their base form and matched against variants.<br>
E.g A charged blowpipe will count as an empty blowpipe.<br><br>
<img width="240" height="78" alt="Context menu with tracking options" src="https://github.com/user-attachments/assets/948b0241-e65a-4120-8d4b-64276b30f482" />

## Bank Filter (optional)
Type `/g` in the bank search box to filter by tracked items.<br>
*Note: shared storage doesn't support custom hints*

## Bank Overlay (optional)
Draw outlines around tracked items in the bank and shared group interfaces.
